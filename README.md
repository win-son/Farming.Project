# Farming Project

A 2D farming game built with Unity.

## Requirements

- **Unity** 6000.3.9f1 or compatible (Editor version from `Farming Project/ProjectSettings/ProjectVersion.txt`)
- Unity packages used: 2D (Aseprite, SpriteShape, IK), Universal RP, Input System

## Getting Started

1. Clone the repository.
2. Open Unity Hub and add this folder (or the `Farming Project` subfolder) as a project.
3. Open the project in Unity.
4. Open **Assets/Scenes/SampleScene** to play.

## Project Structure

- **Assets/Scenes/** – Game scenes (e.g. `SampleScene.unity`)
- **Assets/Scripts/** – C# scripts (e.g. `PlayerController.cs`)

## Controls

Player movement is driven by the new Input System; bindings are configured in the project’s Input Actions asset.
